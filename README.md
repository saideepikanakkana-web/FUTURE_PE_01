📄 AI Website Copy Generator for Local Businesses
🚀 Project Overview
This project is part of the Future Interns – Prompt Engineering Task 1 (2026).
The goal is to design a structured prompt system that generates high-quality, conversion-focused website copy for local businesses using AI tools like ChatGPT.
Many local businesses struggle with:
unclear website messaging
weak value propositions
poor call-to-action (CTA)
generic or outdated content
This project solves that problem using AI-powered prompt engineering.
🏢 Business Chosen
Business Name: GlowUp Beauty Salon
Type: Beauty Salon
Location: Vizag (Visakhapatnam)
Target Audience: Women (18–40), bridal and regular beauty services
🎯 Why This Business?
Local salons often rely on word-of-mouth and have poorly written or no website content.
Improving their website copy can:
attract more customers
build trust
increase bookings
🧠 Prompt System Design
This project uses a modular prompt framework to generate different sections of a website.
🔹 1. Master Prompt
Defines:
role (professional copywriter)
tone and style
business details
writing guidelines
Ensures:
non-generic content
human-like tone
benefit-focused messaging
🔹 2. Homepage Prompt
Generates:
headline (value proposition)
sub-headline (audience + benefit)
introduction
trust points
CTA
👉 Focus: First impression + clarity + engagement
🔹 3. Services Page Prompt
Generates:
service names
descriptions
benefits
👉 Focus: Clear explanation + customer value
🔹 4. CTA Prompt
Generates:
booking CTA
trust-based CTA
urgency-based CTA
👉 Focus: Increasing conversions and actions
🔹 5. Tone Adaptation Prompt
Allows rewriting content in:
friendly tone (salon/cafe)
professional tone (clinic/agency)
👉 Makes system reusable for multiple businesses
